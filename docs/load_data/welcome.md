## Texture Analysis Application

Welcome to our Texture Analysis Application, a tool designed to enhance the speed, interactivity, and intuitiveness of texture database analysis. The application is organized into several tabs, each serving a unique purpose and equipped with its own instructions. For optimal results, we recommend navigating through the tabs in sequential order and adjusting settings as necessary.

To begin, you'll need to load your data file, which should be in either Comma Separated Values (`CSV`) or Excel Spreadsheet (`XLSX`) format. To do this, click on the 'Browse' button and select the appropriate file from your system.

If your file is in `CSV` format, you'll need to specify the symbol used to separate columns. By default, this is set to a comma (`,`). Please note that a period (`.`) must be used as the decimal separator. This is particularly important for users who typically use German data formatting conventions.

For data sets provided as Excel spreadsheets (`XLSX`), the application will automatically assume that the data is located on the first sheet.

We hope you find this application useful and intuitive. Happy analyzing!
